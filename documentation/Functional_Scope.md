# Functional Scope

## Project Name
AI-Powered Customer Support Ticketing System

## Purpose
The project aims to automate customer support operations using Salesforce by managing support tickets efficiently.

## Scope
- Create and manage customer support cases.
- Track ticket status and priority.
- Automate email notifications using Salesforce Flows.
- Generate reports for support analysis.
- Provide an easy-to-use Lightning App interface.

## Users
- Customer Support Agents
- Support Managers
- Salesforce Administrators

## Expected Outcome
The system reduces manual work, improves response time, and enhances customer satisfaction through automation.
